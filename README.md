## Bucket Hat Pattern Generator

A web application that allows users to generate custom sewing patterns of bucket hats based on their measurements. The app features an interactive 3D preview that dynamically updates to reflect user inputs, providing a real-time visualization of the final design. The app also features cloth simulation to give further understanding of the characteristics of the generated hat. The sewing pattern can be exported as a print-ready PDF, making it easy for users to bring their patterns to life.

### Tech
- Next.js framework
- Developed algorithms to generate calculations hat dimensions based on parameters
- SVG patterns were generated with D3.js
- PDF generation was done with jspdf, and SVG conversions with svg2pdf plugin
- 3D mesh was generated and rendered with THREE.js
- Feedback database on Firebase
- Tooltips were created with react-tooltips
